# Device Manager 设备管理器

## 项目简介
这是一个设备管理系统，主要用于千牛账号和设备的管理。项目包含了千牛账号的处理功能。

## 主要功能
- 千牛账号管理
- 设备管理与监控
- 广播系统集成

## 技术栈
项目使用 Python 开发，具体依赖见 requirements.txt 文件。

## 开发环境
- Python 3.10
- 操作系统：Windows
